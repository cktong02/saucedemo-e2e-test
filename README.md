# saucedemo-e2e-test
> A sample end-to-end test against a demo website

## Installation
Install NPM dependencies:
```
$ npm install
```

## Test execution
*Note:* In `.feature` files, scenarios tagged with `@ignore` will not be running.
### Run tests locally with headless browser
```
$ npm run test
```

### Run tests on production env
```
$ npm run test-prod
```
